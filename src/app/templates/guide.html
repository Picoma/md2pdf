Guide d’utilisation

Ceci est votre guide ! Préformaté en HTML, dédié au workflow de votre entreprise.

Utilisation basique

Ce guide est valable dans la plupart des cas d’utilisations : il est conçu pour être le plus simple possible.

  • Aller dans l’onglet “Templates” pour choisir un template.
  • Compléter ce template
  • Aller dans l’onglet “Conversion” et téléverser les fichiers à convertir
  • Cliquer sur “Convertir” : vos PDF sont prêts !

Guide spécifique

Template spécifique

Par exemple, si vous offrez un template avec une utilisation spécifique (changer le nom de fichier pour intégrer un code d’identification, ajouter à la main une figure, …), ces instructions pourraient aller ici. A nouveau, c’est vous le maître !

Intégration

Pour, les fichiers sont envoyées via une requête POST, il doit donc être possible de l’intégrer à une chaine de CI/CD pour créer des documents à la volée d’un commit Git.
